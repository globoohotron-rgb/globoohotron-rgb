# Hey, I'm Volodymyr

I build systems that make AI agents work autonomously — not as chatbots, but as disciplined engineering pipelines with quality control, state machines, and self-correction.

## What I Build

**[Autonomous Orchestrator](https://github.com/globoohotron-rgb/autonomous-orchestrator)** — A TypeScript state machine that drives AI coding agents through a complete product lifecycle. 6 phases, 34 steps, 3 independent quality systems (POKA-YOKE, JIDOKA, Technical Censure). Zero npm dependencies. Includes a daemon with closed-loop control, DAG-based task queue, and a VS Code extension.

**[BillingCore](https://github.com/globoohotron-rgb/billingcore)** — Self-hosted SaaS billing platform built *by* the orchestrator above. Fastify + Prisma + PostgreSQL + Redis + React. Full invoice lifecycle, Stripe payments, subscriptions, audit log, API key auth. Docker-ready.

**[Invoice Dashboard](https://github.com/globoohotron-rgb/invoice-dashboard)** — Full-stack invoice management app. TypeScript end-to-end. [Live demo](https://invoice-dashboard-olive.vercel.app/)
